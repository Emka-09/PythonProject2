Combining Two Numbers
1. О проекте

Это простая программа на Python, которая объединяет два числа в одно.

2. Что делает программа
пользователь вводит два числа
программа объединяет их
выводит результат

Пример:
12 и 34 → 1234

3. Как работает

Числа вводятся как строки, потом они просто соединяются между собой.

4. Технологии
Python
базовый ввод и вывод
5. Как запустить
Открыть файл с кодом
Запустить через Python
Ввести два числа
Посмотреть результат
